Exemplo de iframe com parâmetro "scrolling"

O scrolling no iframe é um atributo que permite controlar a visibilidade da barra de rolagem dentro do elemento <iframe>;
Neste exemplo usamos o "no" para não permitir a rolagem do conteúdo.


Exemplo de iframe com parâmetro "frameborder"

O {frameborder} só aceita dois valores = 1 para com bordas; 0 para sem bordas, ou podemos configurar esta borda por meio da CSS usando a propriedade "border: ; Neste exemplo mudamos a cor da borda."
