Com flex-shrink definido com valor = 1 determinamos o quanto o flex-item vai encolher em relação aos demais, e Com flex-grow definido com valor = 0 determinamos que o flex-item não vai crescer em relação aos demais

imagem 8

Com flex-shrink definido com valor = 1 ; Com flex-grow definido com valor = 1 dizemos que os flex-itens vão crescer para caber na dimensão do contêiner se este ultrapassar o tamanho da soma dos flex-basis.

imagem 9

Com flex-shrink definido com valor = 1 ; Com flex-grow definido com valor = 1 dizemos que os flex-itens vão diminuir mesmo que a dimensão do contêiner seja inferior , ao tamanho da soma dos flex-basis.

imagem 10

Com flex-shrink definido com valor = 0 ; Com flex-grow definido com valor = 1 dizemos que os flex-itens vão manter a dimensão definida se o contêiner tiver dimensão igual ao tamanho da soma dos flex-basis.

imagem 11

Com flex-shrink definido com valor = 0 ; Com flex-grow definido com valor = 1 dizemos que os flex-itens vão quebrar(vazar do contêiner) se a dimensão definida do contêiner tiver dimensão inferior ao tamanho da soma dos flex-basis.



Comportamento padrão dos flex-item em relação ao flex-container.

Considere o comportamento padrão do exemplo abaixo que tem como Configuração, o padrão determinado pelos navegadores da seguinte forma:

.conteiner {
  display: flex;
  /* flex-direction: row; Padrão */
  /* flex-wrap: nowrap; Padrão */
  flex-flow: wrap nowrap; /* Padrão referente a flex-direction e flex-wrap.*/
}
.item {
  /* flex: auto; Removido */
  flex-basis: auto;/* Padrão */
  flex-shrink: 1;/* Padrão */
  flex-grow: 0;/* Padrão */
}
