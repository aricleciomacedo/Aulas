	ENVELOPAMOS com a tag <header> o conteúdo do cabeçalho e também envelopamos o <a> link(âncora) para o site do auto do projeto; Adcionamos o Título principal do conteúdo <h1> e o parágrafo <p>.

		O que é target _blank?
			O target="_blank" é um atributo do HTML que abre um link em uma nova janela ou aba do navegador.
	ENVELOPAMOS com a tag <section> as 5 seções de nosso conteúdo e adcionamos as BraksHovers <br> para quebrar linha entre as frases e como sections não separa por parágrafos, então envelopamos cada uma das septilhas com a tag <p>.

	CRIAMOS a pasta estilos e o arquivo .css
	COMO?
		Abaixo da tag <title>, digitamos link:css e teclamos ENTER;
			VIDE = <link rel="stylesheet" href="./estilo/style2.css">
		Paramos a seta do mouse sobre "style.css" , então clicamos em "Siga o Link(CTRL + clique)" e clicamos em "criar o arquivo" para criar o (arquivo) e a pasta para estilo.

	POR FIM; COm o arquivo "style.css" aberto adcione a regra padrão = @charset "UTF-8" que vai servir para aceitar todos os caracteres de acentuação de textos em vários idiomas.

	+ SOBRE:
	@charset "UTF-8" 

		A regra @charset "UTF-8" em CSS serve para especificar a codificação de caracteres UTF-8 na folha de estilo: 
		Garante a decodificação dos caracteres, mesmo que eles não sejam os mesmos na folha de estilo 
		É uma boa prática e altamente recomendável 
		É uma questão de segurança 
		A regra @charset deve ser o primeiro elemento na folha de estilo e não ser precedida de qualquer caráter. Além disso, é importante salvar a folha de estilo na codificação de caracteres UTF-8. 
		O UTF-8 (UCS Transformation Format 8) é a codificação de caracteres mais comum da World Wide Web.

	Nesta aula é ensinado a usar o recurso zebrado que serve para melhorar a visibilidade dos dados da tabela, pois permite pintar uma linha de cada cor.

SAIBA que se você fizer o seletor CSS para todo <tr> filho direto do <tbody> da seguinte maneira "tbody > tr {}", não ficará do jeito esperado que é o = zebrado.

ENTÃO para melhor efeito, usamos a pseudo-classe que faz aplicação da propriedade que queremos de maneira pré-defenida.
	EX: Se queremos Cor de fundo, mas queremos pintar uma sim e outra não, e assim seguir, então a pseudo-classe ":nth-child(2n) {background-color: gray;}" fará esse efeito.

OUTRA maneira é adicionar "odd" = "2n-1" = ímpar, ou "even" = "2n" = par, na pseudo-classe indicando que queremos manipular os elementos ímpares ou pares.

VALE lembrar que podemos usar o seletor com a pseudo-classe duas vezes, e para usar em ambas os valores para células ímpares e pares com diversas propriedade, mas só faz sentido neste caso se for usar para o efeito ensinado.

	Iníciamos esta aula com um LEMBRETE:
		O W3C sempre recomenda o uso de "scope" nas <th>´s, para que se define se ele está em linha ou em coluna.
	
	AGORA vejamos mais sobre "Grupos de Linhas e Gurpos de Colunas".
	1- scope="colgroup" é definido quando existem mais de uma coluna contendo dados.
	 1a- scope="col"	é definido quando existe apenas uma coluna contendo dados.
	2- scope="rowgroup" é definido quando existem mais de uma linha contendo dados.
	 2a- scope="row"	é definido quando existe apenas uma linha contendo dados.

	SAIBA:
		Visão geral criada por IA

			O scope rowgroup ou colgroup são atributos HTML que indica que uma célula de cabeçalho se aplica a um grupo de linhas ou colunas: 
			
			1- O atributo scope especifica se uma célula de cabeçalho é um cabeçalho para uma coluna, linha, ou grupo de colunas ou linhas.

			>>>> Os valores possíveis para scope são row, col, rowgroup, e colgroup .
				O valor rowgroup indica que a célula de cabeçalho se aplica a um grupo de linhas.

			>>>> O atributo scope é usado para: 
				Associar células de cabeçalho com células de dados em tabelas complexas 
				Clarificar o escopo de qualquer célula usada como cabeçalho 
				Ajudar leitores de tela e outras tecnologias assistivas a entenderem a relação entre células de cabeçalho e células de dados 
				O atributo scope não tem efeito visual em navegadores web normais, mas pode ser usado por leitores de tela.

Vídeo:
Agrupando Colunas com colgroup - @Curso em Vídeo HTML5 e CSS3
Link:
https://www.youtube.com/watch?v=1Ro7rjINNb0&list=PLHz_AreHm4dmcAviDwiGgHbeEJToxbOpZ&index=38

	NEsta aula criamos a tabela 005 (arquivo "tabela005.html") na pasta de exercícios "ex0023".
	
	VEJA também a mesma tabela e alguns cometários adcionais no arquivo "index.html" e seu estilo está no arquivo "style2.css".

	1- Para aplicar estilo em linha de tabela, o procedimento é mais fácil, pois podemos atribuir uma "class" nas <tr>´s ou <tag> que queremos estilizar e por meio de seu seletor CSS, podemos aplicar as configurações que queremos.

	2- Para aplicar estilo em colunas, o procedimento é um pouco diferente.
		OBS:
			Há quem atribua a "class name" a cada <th>, <td> da coluna, tornando isso mais trabalhoso e possívelmente dará dificuldades para manutenção do código se a tabela for complexa com maior dúmero de dados(registros).

		PARA ISSO, na HTML5 existe um nova"<tag>" a <colgroup> que proporciona criar uma <tag = col> para cada coluna, e assim poder aplicar um único estilo para várias colunas de tabela.
		VEJA no index a aplicação do exemplo, onde cada <col> representa um coluna:
			  <table>
				<colgroup>
			  		<col>
			  		<col>
			  		<col>
			  		<col>
			  	</colgroup>
			  </table>
*		DETALHE:
			"Podemos atribuir diferentes classes ou (id´s) para cada <col>, e assim poder configurar cada coluna de forma única e particular."

	3 - Podemos apliar expanção para duas colunas de uma vez, por meio de class + span.
			No exemplo da aula fizemos:
				<col class="cname">	
				<col class="cgroup" span="2">
				<col class="cprof">
			COM ISSO dizemos que duas colunas , tem class-name iguais e receberão o mesmo estilo.
			O "span" neste caso corresponde ao número de colunas que a classe corresponde.

	4- Podemos ainda fazer de exemplo da seguinte maneira:
		HTML:
		<colgroup>
			<col class="yellow">
			<col class="red">
			<col class="yellow">
			<col class="red">
		</colgroup>

		CSS:
		col.yellow {
			background-color: yellow;
		}
		col.red {
			background-color: red;
		}

		Assim as <col> de class=yellow receberão a cor amarela e as <col> de class=red receberão a cor vermelha; VEJA no index o exemplo nas tags <th>OBS</th>
